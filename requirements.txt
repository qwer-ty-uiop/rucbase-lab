# JOIN功能需求文档

## 1. 功能概述

实现完整的SQL JOIN功能，支持以下JOIN类型：
- INNER JOIN（内连接）
- LEFT JOIN（左外连接）
- RIGHT JOIN（右外连接）
- 支持ON条件表达式

## 2. 功能需求

### 2.1 支持的JOIN类型

#### 2.1.1 INNER JOIN
```sql
SELECT * FROM t1 INNER JOIN t2 ON t1.id = t2.id;
SELECT * FROM t1 JOIN t2 ON t1.id = t2.id;  -- INNER可省略
```

#### 2.1.2 LEFT JOIN
```sql
SELECT * FROM t1 LEFT JOIN t2 ON t1.id = t2.id;
SELECT * FROM t1 LEFT OUTER JOIN t2 ON t1.id = t2.id;  -- OUTER可省略
```

#### 2.1.3 RIGHT JOIN
```sql
SELECT * FROM t1 RIGHT JOIN t2 ON t1.id = t2.id;
SELECT * FROM t1 RIGHT OUTER JOIN t2 ON t1.id = t2.id;  -- OUTER可省略
```

### 2.2 ON条件支持

支持在ON子句中使用以下表达式：
- 比较运算符：=, <, >, <=, >=, <>
- 逻辑运算符：AND, OR, NOT
- 字段引用：t1.id, t2.name

### 2.3 多表JOIN

支持多表连接：
```sql
SELECT * FROM t1 
INNER JOIN t2 ON t1.id = t2.id 
LEFT JOIN t3 ON t2.id = t3.id;
```
